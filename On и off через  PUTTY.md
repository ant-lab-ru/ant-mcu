# Результат:
- [ ] Включение и выключение светодиода вынесено в отдельные функции
- [ ] При вводе в терминал символа ```e``` - светодиод  включается, при вводе ```d``` - светодиод выключается
# Конфигурация проекта
- Создайте проект c названием ```02-on-off```
- Скорректируейте ```CMakeLists.txt```
# Прошивка
- Подключите библиотеки из pico SDK для взаимодействия с потоком ввода/вывода и с микроконтроллером
- Реализйте функции включения и выключения светодиода:
``` c
void led_enable(uint pin) {
	// Код реализующий логику включения светодиода
}
void led_disable(uint pin) {
    // Код реализующий логику выключения светодиода
}
```
- Проинициализируйте ```stdio``` и ```gpio``` вывод
- Добавьте в тело бесокнечного цикла считывание символа из терминала
- Добавьте  логику обработки считанного символа
``` c
        switch(symbol)
        {
            case 'e':
	            // Включение светодиода
                break;
            case 'd':
				// Выключение светодиода
                break;
            default:
                // Предупрежедние о неккоректном вводе
                break;
        }
```
