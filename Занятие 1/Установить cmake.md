- Linux. Выполнить в терминале:
``` bash
sudo apt update
sudo apt install cmake
cmake --version
```

- MacOS. Выполнить в терминале:
``` bash
brew install cmake
cmake --version
```

- Windows:
	- Скачать установщик: https://cmake.org/download/
	- Запустить установщи, обязательно отметить добавление cmake в Path
	- После установки проверить ```cmake --version```
