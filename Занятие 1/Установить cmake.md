- Linux. Выполнить в терминале:
``` bash
sudo apt update
sudo apt install cmake
cmake --version
```

- MacOS. Выполнить в терминале:
``` bash
brew install cmake
cmake --version
```

- Windows:
	- скачать установщик: https://cmake.org/download/;
	- запустить установщик, обязательно отметить добавление cmake в Path;
	- после установки проверить ```cmake --version```.
