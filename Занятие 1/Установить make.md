- Linux. Выполнить в терминале:
``` bash
sudo apt update
sudo apt install make
make --version
```

- MacOS. Выполнить в терминале:
``` bash
brew install make
make --version
```

- Windows:
	- Скачать установщик: https://gnuwin32.sourceforge.net/packages/make.htm
	- Запустить установщик, (обязательно отметить добавление git в Path, если есть)
		- В случае если установщик был, запущен, а cmake все еще оказался не установленным:
		-  Найдите путь к `make.exe`: Убедитесь, что вы знаете, где установлен make (скорее всего`C:\Program Files\GnuWin32\bin`).
		- Откройте Переменные среды:
	    - Нажмите `Win + R`, введите `sysdm.cpl` и нажмите Enter.
	    - Перейдите на вкладку **Дополнительно** и нажмите **Переменные среды...**.
		- **Измените Path**:
	    - В разделе **Системные переменные** найдите переменную **Path** (или **PATH**) и нажмите **Изменить**.
	    - Нажмите **Создать** и вставьте путь к папке, содержащей `make.exe`.
	    - Нажмите ОК во всех окнах.
	- После установки проверить ```make --version```
