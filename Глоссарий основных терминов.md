**Микроконтроллер (МК, MCU)** — компьютер на одной микросхеме (процессор + память + периферия)

**Встраиваемая система**

**Процессор (CPU)** — вычислительное ядро, выполняет инструкции программы

**Flash-память** — энергонезависимая память, хранит прошивку

**RAM (ОЗУ)** — оперативная память, хранит рабочие данные

**Периферия** — аппаратные модули для ввода-вывода (GPIO, UART, SPI, I2C, таймеры, АЦП)

**GPIO** — цифровые входы/выходы общего назначения

**Прошивка (firmware)** — программа для микроконтроллера, работающая без ОС

**Компилятор** — программа, переводящая код на Си в машинный код

**Кросс-компиляция** — компиляция на одной платформе для выполнения на другой

**SDK** — набор инструментов и библиотек для разработки (драйверы, примеры, система сборки)

**Программатор** — устройство для записи прошивки во Flash

**Datasheet** — технический паспорт микросхемы с характеристиками
