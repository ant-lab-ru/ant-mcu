# Программа курса

1. **Моргающий светодиод**
	1. Сущность микроконтроллера (документация)
	2. SDK
	3. Система сборки
	4. Бинарный файл
	5. Программатор
	6. Запуск прошивки
2. **Управляемый светодиод**
	1. Внешнее управление прошивкой
	2. Сущность пользователь
	3. Сущность прибора и его API
	4. Канал коммуникации "Прибор <-> Пользователь"
	5. Реализовать прошивку прибора
	6. Воспользоваться прибором (терминал, Pyhton-скрипт)
	7. Реализовать функции mem() и wmem()
3. **Измерительный прибор**
	1. Понятия процессора и периферии МК
	2. Концепция настройки периферии
	3. Архитектура суперцикла и концепция задач
	4. Задача измерения напряжения
	5. Задача реализации API
	6. Задача отправки телеметрии
4. **Интерфейсы передачи данных**
	1. Поддержка API сторонних устройств
	2. Интерфейсы взаимодействия SPI, I2C, UART, CAN, Ethernet, USB, SpaceWire, I2S
	3. *Обмен информацией по интерфейсам*
	4. Концепция драйвера
	5. Концепция библиотеки

6. **Текст на дисплее**
