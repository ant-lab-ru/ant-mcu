# Конфигурация проекта
- Создайте проект c названием ```02-echo```
- Скорректируейте ```CMakeLists.txt```
# Прошивка
- Подключите библиотеки из pico SDK
- Проинициализируйте ```stdio```
- Добавьте в тело цикла считывание введенного символа из PUTTY терминала:
``` c
char symbol = getchar();
```
- Добавьте вывод в терминал считанный символ и его код в ```ASCII```
